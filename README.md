## Overview

Built for Lapis Group to help Afghan students discover and access higher education abroad. Aggregates opportunities from universities worldwide into a single searchable, filterable interface — pulling live data from an Excel source.

## Features

**Search & Filter**
- Full-text search across institutions and details
- Filter by format, degree type, scholarship availability
- Filter by cost range and accreditation status
- Instant results with animated card transitions

**Opportunity Cards**
- Institution name, degree type, and format at a glance
- Scholarship and fee indicators
- Estimated cost display
- Direct apply button with loading feedback

**UI & Experience**
- Responsive grid layout (mobile-friendly)
- Sticky navigation with blur effect on scroll
- Animated number counters on stats
- Scroll-triggered card entrance animations

## Tech Stack

**Frontend:** HTML5, CSS3, Vanilla JavaScript  
**Data:** Excel (.xlsx) via SheetJS — loaded dynamically at runtime  
**Icons:** Font Awesome 6  
**Fonts:** Inter (Google Fonts)  
**Hosting:** Hostinger (Production)

## Deployment

- **Platform:** Hostinger Shared Hosting  
- **Requirements:** Static file hosting, no backend needed  
- **Data Source:** `Opportunities at Universities for Afghans.xlsx` (loaded client-side)  
- **Timeline:** Developed & Deployed Mar 2026
